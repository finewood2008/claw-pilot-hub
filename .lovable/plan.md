

# Q-CLAW 官网首页重建计划

## 概述
根据参考图片，创建一个深色风格的 Q-CLAW 产品官网首页，替换当前的空白 Index 页面，并将路由 `/` 指向新首页（而非重定向到登录页）。右上角"控制台"按钮链接到 `/login`。

## 页面结构（从上到下）

### 1. 顶部导航栏
- 左侧：Q-CLAW Logo + 品牌名
- 中间：功能介绍 / 下载中心 / GitHub 链接
- 右侧：「控制台」按钮 → 链接到 `/login`

### 2. Hero 区域（深色渐变背景）
- 顶部标签："限时福利，新用户注册赠送 300万 Token"
- 大标题："Q-CLAW" + "让每个人都能轻松驾驭 AI 助手"（AI 助手使用渐变色高亮）
- 副标题描述文字
- 两个 CTA 按钮：「下载 Q-CLAW 客户端」（主色）和「GitHub 源码」（描边）

### 3. 核心特性区域
- 标题："核心特性" + 副标题
- 三列卡片：
  - 免配置安装（闪电图标）
  - 人民币直充（钱包图标）
  - 多端无缝连接（设备图标）
- 每张卡片含图标、标题、描述文字

### 4. 运行原理说明区域
- 标题："Q-CLAW 运行原理说明"

  - 底部标注"自建安全开源"

### 5. 多端渠道生态矩阵区域
- 标题 + "前往下载中心" 按钮
- 即时通讯渠道卡片（主流 IM 平台直连）：QQ / 微信 / 企业微信 / 飞书 / 钉钉 / Slack / Discord / Telegram / WhatsApp
- 三列底部卡片：小程序矩阵 / 原生 APP / 硬件组件

### 6. 页脚
- 版权信息："© 2026 Q-CLAW Platform. Powered by OpenClaw."

## 技术实现

### 文件变更

1. **`src/pages/Index.tsx`** — 完全重写
   - 创建完整的深色主题官网首页组件
   - 使用 Tailwind CSS 实现深色背景（bg-[#0a0e1a] 等深色调）
   - 使用 lucide-react 图标替代原图中的图标
   - 所有内容纯 CSS/HTML 实现，不嵌入截图
   - 响应式设计，支持移动端

2. **`src/App.tsx`** — 修改路由
   - 将 `path="/"` 从 `Navigate to /login` 改为渲染 `<Index />` 组件
   - 导入 Index 页面

### 样式策略
- 页面独立使用深色配色（不依赖全局 dark mode），通过内联 Tailwind 类实现
- Hero 区域使用紫色/蓝色渐变背景
- 卡片使用半透明深色背景 + border
- "AI 助手"文字使用渐变色（从蓝到紫/粉）
- 架构图使用 CSS flexbox/grid + border/圆形元素复刻

### 关键交互
- 「控制台」按钮 → `<Link to="/login">`
- 「下载 Q-CLAW 客户端」和「GitHub 源码」按钮 → 可设为 `#` 占位链接
- 「前往下载中心」→ 可设为 `#` 占位链接
- 导航栏链接 → 页内锚点滚动（功能介绍 → #features，下载中心 → #channels）
