### 具体改动

**文件：`src/pages/Index.tsx`**（架构图板块 lines 166-305）

1. **第1层 - LLM 大模型服务层**（保留现有内容，样式微调）
   - 保持现有 6 个大模型卡片（Gemini、OpenAI、Qwen、Claude、DeepSeek、Llama）
   - 框样式保持现有的 `rounded-2xl border border-white/5 bg-[#111827]/40`
   - 保留 "LLM 大模型服务层" 标题

2. **层间连接线**
   - 每两层之间用居中的垂直虚线（`border-dashed`）+ 小箭头连接
   - 复用现有的 `animate-dash-flow` 动画效果
   - 高度约 `h-8`，与当前 LLM→Agent 的连接线一致

3. **第2层 - Q-CLAW API 管理平台层**（新独立框）
   - 从现有代理框中提取 API 管理平台节点
   - 单独用一个与第1层同样风格的框包裹
   - 内部居中显示：Cloud 图标 + "Q-CLAW API 管理平台" + 副标题
   - 移除代理外框标签（"Q-CLAW Agent 代理层"），改为框内标题

4. **第3层 - 接入终端与 OPENCLAW 本地运行层**（新独立框）
   - 将现有的「接入终端 CLIENTS」和「OPENCLAW Core Engine」放入同一个框
   - 框标题为 "接入终端与 OPENCLAW 本地运行层"
   - 内部布局：左侧为接入终端卡片网格，右侧为 Core Engine 圆形组件
   - 中间保留 "Channel 通道直连" 标签和箭头

5. **移除旧的三角拓扑 SVG 连线**
   - 删除 `<svg>` 中的三条斜线（API→Clients、API→Engine、Clients→Engine）
   - 删除绝对定位布局（`md:absolute`），改为正常文档流

   - 三层在桌面和移动端均为垂直堆叠（自然流式布局）
